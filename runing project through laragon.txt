1. Download laragon full version
https://laragon.org/
2. right click root, www, Switch Document Root and select the github repository.
3. click start all
4. click database, open Laragon.MySQL, under file click load sql file, choose "DB" found in D4G folder.
5. file encoding is ok
5. execute sql (you have to run it)
6. click web to run local host
7. click HTML-PHP
8. Now all account info will be stored in the DB
9. if you moddified the database, delete DB.sql in root and then export registration by rightclicking registration and clicking export database as sql. see screenshot on discord (notes-resrouces channel) for settings.
